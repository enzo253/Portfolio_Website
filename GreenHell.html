Green Hell Predictor

The Green Hell Predictor is a performance forecasting web application that uses AI and advanced analytics to predict missing vehicle specifications and Nürburgring lap times, helping users explore and compare car metrics intuitively.

🛠️ Tech Stack

  • Python: Data processing and modeling using Pandas, NumPy, Plotly, and Requests
  • Streamlit: Lightweight framework for building dynamic and interactive web apps
  • PostgreSQL: Database storage and integration for car datasets
  • Together API: AI-based prediction model integration for missing specs and lap times

⚙️ Process & Methodology

The project started by gathering and cleaning comprehensive car data. I then trained machine learning models to predict missing car attributes and estimate Nürburgring lap times. The web app was built with Streamlit for a fast, lightweight UI that integrates car comparisons, acceleration curves, and 3D performance visualizations.

The final product allows users to interactively select cars, view AI-predicted specs, compare vehicles across multiple metrics, and explore performance graphs — all optimized for desktop and mobile.

📈 Outcomes

  • 🏎️ Built a dynamic car performance prediction tool with live graph updates
  • 📈 Visualized acceleration curves and performance stats with 3D interactive plots
  • 🧠 Integrated AI predictions for missing vehicle attributes to expand dataset usability
  • 🚀 Laid the groundwork for future integration of user-uploaded car data and model refinement
